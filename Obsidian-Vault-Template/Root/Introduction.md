# Hotkeys
##### Obsidian Def
1. ctrl + q = quick-search of all notes
2. ctrl + n = new note
3. ctrl + t = new tab
4. ctrl + w = close tab
5. ctrl + tab = select right tab
6. ctrl + shift + tab = select left tab
7. ctrl + click on a link in [[Markdown]] opens it in a new tab, and also shows preview window
8. alt + left arrow = go back to the previous window's note, right arrow goes forward to selected note
9. ctrl + e = toggles between edit mode and preview mode
10. ctrl + p = print note to pdf
11. 'ctrl + f' = opens word-search
12. ctrl + '+' zoom, '-' for zoom-out
13. 'ctrl + r' to run code
14. 'shift + del' = delete line
15. 'ctrl + shift + del' = delete note
---
#### Custom Obsidian
- ctrl + shift + t = new note from template
- ctrl + shift + w = close all tabs but selected tab
- ctrl + shift + z = undo closed tab
- ctrl + , or . = toggle corresponding side-panel
- ctrl + / = split current window to side
- alt + s = settings
- alt + ; = calendar
- alt + ' = new-event

- Modifies current mouse-place, or selected text
	- ctrl + i = italics
	- ctrl + b = bold
	- ctrl + u = underline
	- ctrl + 1 = embedded note
		- ![[Markdown]]
	- ctrl + 2 = hyperlink to website
	- ctrl + 3 = numbered list / disabled list
	- ctrl + 5 = hidden text
	- ctrl + 6 = headers
	- ctrl + 7 = toggle list / checklist
	- ctrl + 9 = open all sublists
	- ctrl + 0 = collapse all sublists
- ctrl + alt + i = super / sub / footnote quickadd like <sup>this</sup> or <sub>this</sub> or <sub>more</sub> <sub><sub>this</sub></sub>

#### PC General Defaults
##### Windows
1. 'wind. + e' = file explorers
2. 'wind. + [d / m]' = desktop
3. 'wind. + a' = audio
4. 'wind. + x' = settings
5. 'wind. + c' = teams contacts
6. 'wind. + v' = clipboard
7. 'wind. + n' = notifications / calendar
8. 'wind. + tab' = display open apps / desktops
9. 'wind. + f' = feedback hub
10. 'wind. + u' = accessibility / settings
11. 'wind. + [1-0]' = display application
12. 'wind. + [arrow]' = manipulate how app is displayed
13. 'wind. + z' = app-display options
14. 'wind. + [; or .]' = emoji
15. 'wind. + , ' = view desktop temporarily
16. 'wind + [+ / -]' = direct magnify
---
##### General
1. 'alt + tab' = scroll application windows
2. 'ctrl + tab' = scroll tabs
3. 'alt + F4' = close program
4. 'ctrl + w' = delete window
5. 'touchpad + three finger up' = display open apps / desktops
6. 'shift + end' = highlight to end of line
